# **App Name**: Habitualize

## Core Features:

- User Authentication: Secure sign-up, login, and logout functionality with email verification and password management.
- Habit Creation & Management: Users can create, edit, delete, and pause habits with customizable details like name, category, frequency, reminders, and priority.
- Daily Habit Tracking: A daily checklist view where users can mark habits as completed, missed, or skipped, with optional notes and automatic progress saving.
- Visual Progress Tracking: Progress bars, streak counters, weekly charts, monthly calendar views, and color indicators provide visual feedback on habit completion.
- Motivational Prompts: The tool helps inspire users with daily motivational quotes, habit tips, and success messages.
- Reporting Dashboard: Dashboard provides progress summary, today's habits, streaks, and quick actions.
- Settings & Customization: Dark/Light mode, data export and other key preferences for the user.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) for trust and focus.
- Background color: Light blue (#E3F2FD), a very desaturated version of the primary.
- Accent color: Muted violet (#7986CB) for highlighting important actions.
- Headline font: 'Space Grotesk' sans-serif for headers and short amounts of body text, paired with 'Inter' sans-serif for longer body text.
- Clean, minimalist icons for habit categories and actions.
- A clean, intuitive layout with a focus on daily habits and progress tracking.
- Subtle animations for task completion and streak milestones.
